Pandas可以支持便捷读取与处理多种来源的数据。可读取本地存储的静态文件，也可以读取存储在远程数据库当中的数据。

Doc Link: https://pandas.pydata.org/pandas-docs/stable/reference/api/pandas.Series.array.html

### Pandas可以读取的一些常见数据格式

1. **CSV 文件**：逗号分隔值文件是一种常见的数据存储格式，Pandas 可以轻松地读取和写入 CSV 文件。
    
2. **Excel 文件**：Pandas 支持读取和写入 Excel 文件，包括 .xls 和 .xlsx 格式。
    
3. **SQL 数据库**：Pandas 可以通过 SQLAlchemy 连接到各种类型的 SQL 数据库，并执行查询操作将数据读取为 DataFrame。
    
4. **JSON 文件**：Pandas 可以读取和写入 JSON 格式的数据。
    
5. **HTML 文件**：Pandas 可以从 HTML 文件中提取表格数据并转换为 DataFrame。
    
6. **Stata 文件**：Pandas 可以读取 Stata 文件（.dta 格式）中的数据。
    
7. **SAS 文件**：Pandas 支持读取 SAS 文件中的数据。
    
8. **Parquet 文件**：Pandas 可以读取和写入 Parquet 格式的数据，这是一种高效的列式存储格式。
    
9. **Feather 文件**：Pandas 可以读取和写入 Feather 格式的数据，这是一种轻量级的二进制列式存储格式。
    
10. **HDF5 文件**：Pandas 支持读取和写入 HDF5 格式的数据，这是一种用于存储大型数据集的高性能文件格式。
